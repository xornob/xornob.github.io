How can you calculate 'Randomness'?

back
  • We start with a string that is 2 characters long.
  • This string can be any of ['00','01','10','11'].
  • If the 0s and 1s are chosen randomly, then all 4 are equally likely.
  • Therefore, if we split a larger string into substrings of length 2, all 4 cases should appear evenly.
  • Even though it seems wrong, random generation is just as likely to generate '0000000000' or '1111111111' as it is to generate '1010011010'.
  • This program works by separating the large string into substrings, and comparing the number of times all of the possibilities show up.
  • Try a sequence generated from RANDOM.ORG (output as 'binary'), or try typing in your own sequence to see how 'random' you are!
